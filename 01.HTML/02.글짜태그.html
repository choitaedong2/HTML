일반 글꼴(태그를 감싸지 않은 텍스트)

b(Bold): 글자를 두껍게 표시하는 태그

strong : 글자를 두껍게 표시하는 태그 + 스크린 리더에서 강조하여 읽어줌

i: 기울임 글꼴

em : 특히 강조할 때 기울임 글꼴로 표시 + 스크린 리더에서 강조하여 읽어줌

mark : 형광펜 효과를 주는 태그(색상은 CSS에서)

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 (Style에서도 지정 가능)

기본글자에 sub : 아래첨자를 넣어주는 태그
sup : 윗 첨자 를 나타내는 태그

abbr: 줄임말 표시
HTML이란 월드 와이드 웹의 마크업 언어이다.

blockquote : 인용문
당신이 할 수 있는 가장 큰 모험은 당신이 꿈구고 삶을 사는 것이다.
