이미지 관련 태그

* src 속성


* alt 속성

- 이미지의 경로가 잘못되었거나 이미지를 제대로 표현하지 못할 경우 대체 텍스트의 용도
- 시각장애인들을 위한 스크린리더에서 이미지를 읽어주는 설명 문구




* width, height 속성

이미지의 가로, 세로 길이 조절 가능
고정길이 (px), 가변길이 (%)로 지정 => 기본값 px단위



고정길이 단위(px) = 기본값 : 화면사이즈가 조정 되어도 이미지 크기는 변경되지 않는다

가변길이 단위(%) : 화면사이즈가 조정되면 그에 따라 이미지의 크기도 변경됨
