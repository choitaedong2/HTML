# VS CODE
### VS CODE EXTENTION
- Open In BROWSER : 브라우저로 창을 바로 실행(단축키 Alt + b)
- Quick HTML Previewer : VS code에서 Preview 화면 생성(단축키: Ctrl + p, s)
- Auto Rename Tag : 태그명 변경시 뒷 태그 자동 변경
- htmltagwrap : 특정 셀렉션을 태그로 감싸줌(단축키 Drag + Alt + W or Option + W)
